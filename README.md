# PakHealth Hospital Analytics Dashboard
This project is a comprehensive Hospital Analytics Dashboard built using Power BI.
The solution transforms hospital operational, financial, and inventory data into actionable insights through interactive visualizations and KPI monitoring.
The dashboard supports data-driven decision-making for hospital administrators, department managers, and finance teams.
## Objectives
- Monitor hospital revenue performance
- Track appointment efficiency
- Analyze doctor and department productivity
- Understand patient demographics
- Optimize inventory management
- Support strategic decision-making
- ## Tools & Technologies
- Power BI Desktop
- Power Query
- DAX
- Excel
- Data Modeling
- Healthcare Analytics
- ## Dashboard Pages
### 1. Executive Performance Dashboard
- Revenue KPIs
- Appointment Metrics
- Revenue Target Tracking
### 2. Department & Physician Performance Analytics
- Department Revenue
- Doctor Efficiency
- Wait Time Analysis
### 3. Patient Demographics & Financial Analytics
- Payer Analysis
- Billing Insights
- Patient Demographics
### 4. Inventory Intelligence & Revenue Forecasting
- Inventory Monitoring
- Expiry Wastage Analysis
- What-if Scenario Modeling
- ## Key Insights
- Cardiology generated the highest revenue contribution.
- Appointment completion rate remained near 80%.
- Revenue declined slightly from 2022–2024.
- Insurance-covered patients generated significant revenue.
- Inventory wastage highlighted overstocking opportunities.
